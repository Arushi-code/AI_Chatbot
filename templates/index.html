Project Modules

What The Website Demonstrates

NLP Preprocessing

Tokenization, lemmatization, and bag-of-words vector generation prepare text for the model.

Deep Learning

A TensorFlow/Keras neural network classifies user messages into intent categories.

Entity Recognition

Lightweight extraction identifies useful values such as email, phone, date, time, name, and topics.

LLM Fallback

OpenAI-powered fallback improves flexible answers when the intent model is not confident.

Flask API

The `/chat` endpoint connects the website to the chatbot engine.

Training Dataset

The editable JSON intent file makes it simple to expand FAQs and retrain the model.
